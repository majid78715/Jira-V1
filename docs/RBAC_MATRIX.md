# RBAC Matrix

| Role | Description | Phase 2 Scope |
| --- | --- | --- |
| SUPER_ADMIN | Full platform control | /api/admin/*, /api/companies mutations |
| VP | Executive stakeholder | Read-only dashboards |
| PM | Product/program manager | Invite project managers, manage projects/tasks, approve onboarding + assignments |
| ENGINEER | Internal delivery | Read-only plus attendance/time tracking on dashboard |
| PROJECT_MANAGER | Vendor lead | Invite developers, submit profile changes, create project tasks, request assignments |
| DEVELOPER | Vendor contributor | Accept invitations, request profile changes, view/complete approved assignments, clock in/out, log time |
| VIEWER | Read-only | Read-only |
